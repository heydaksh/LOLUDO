# LOLUDO

LOLUDO is a fun and modern and extra ordinarry version of the classic Ludo  game built using Flutter. The game provides a smooth gameplay experience with simple visuals and engaging mechanics and fun to use extra features which are exclusively devleoped to engage fans.

## Features

- Classic but not as all Ludo gameplay
- Play with friends on the same device
- Smooth and optimized gameplay
- Simple and colorful user interface
- Lightweight and responsive design

## Tech Stack

- Flutter
- Dart
- Provider / State Management
- Custom animations and game logic

## Getting Started

1. Clone the repository

```
git clone https://github.com/heydaksh/LOLUDO.git
```

2. Navigate to the project folder

```
cd LOLUDO
```

3. Install dependencies

```
flutter pub get
```

4. Run the app

```
flutter run
```

## Project Purpose

This project was developed to practice Flutter development, game logic implementation, state management, and performance optimization.
